# Evidence: W3C Time Temporal Model

Status: blocked.

Tracked artifacts:

- `spec.md`
- `plan.md`
- `metadata.json`
- `index.md`

Dependency boundary:

- Party attribution is available only as a temporal derived layer with
  validation constraints.
- Member identity is available as a validated local review package, not yet as
  a corpus-wide release surface.
- Sitting and proceeding reconciliation remains incomplete, so periods and
  office intervals would be premature as a published model.
- RDF linked-data is still sample-not-release and records W3C Time only in the
  sample mapping notes.

Planned scope:

- interval and instant representation
- parliamentary periods, sittings, offices, and memberships
- open intervals and precision rules
- overlap and provenance checks
- RDF-ready mappings and validation fixtures
