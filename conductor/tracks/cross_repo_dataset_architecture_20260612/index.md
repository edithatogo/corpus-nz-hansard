## Corpus Boundaries

| Corpus | Domain | Primary Source |
|---|---|---|
| corpus-nz-hansard | Parliamentary debates | DocumentsDB, Bills API, HathiTrust, Wikipedia, Wikidata, Parliament website, Electoral Commission |
| corpus-law-nz | Enacted legislation | NZ Legislation API (key), NZLII |
| corpus-nlp (planned) | NLP outputs | Derived from Hansard and Legislation |
