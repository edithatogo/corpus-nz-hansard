# Track Researcher Client Helpers

## Purpose

Provide read-only helper examples for local analysis of document-level and RDF sample
artifacts.

## Supported Artifacts

- `samples/researcher-client-helpers/hansard-mini.csv`
- `samples/rdf-linked-data/linked-data.ttl`

## Validation

- Python and DuckDB examples run against the bundled sample CSV.
- The RDF sample summary runs against the maintainer-review sample graph.
- R and standalone SPARQL examples are deferred until RDF endpoint release artifacts
  are ready.

## Deferred Helpers

- R example surface
- Standalone SPARQL example surface
