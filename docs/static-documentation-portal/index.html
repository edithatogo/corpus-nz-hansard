Release Ladder

The ladder separates document-level release, authority-source evidence, neutral components, endpoints, and upstream contribution packages.

LevelIDStatusDescription
1document-levelpublishedCanonical normalized document records from the supplied DocumentsDB extract, plus release metadata, provenance, and public-surface evidence.
2authority-sourcevalidated-manifestOfficial-first source inventories and retrieval evidence used to support derived member, party, sitting, bill, motion, vote, and procedure assertions.
3neutral-componentvalidated-manifestValidated repository-owned component tables such as sittings, proceedings, speech turns, members, parties, motions, votes, bills, topics, and annotations.
4endpointplannedGenerated standard-specific endpoint artifacts such as ParlaMint/TEI, Popolo/Open Civic Data, Akoma Ntoso, CAP/ParlaCAP, CoNLL-U, RDF, Croissant, RO-Crate, Frictionless, DCAT, and PROV-O.
5upstream-contributionplannedReviewed samples, fixtures, mapping notes, issue-ready evidence, or contribution packages for external maintainers and standards communities.
